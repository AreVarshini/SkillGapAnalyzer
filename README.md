# Skill Gap Analyzer & Career Recommendation System

## Project Overview
This project is a rule-based career guidance system developed using Java, Spring Boot, and MySQL. It analyzes user skills, compares them with job role requirements, identifies missing skills, and evaluates the user's readiness level for a particular career path.

The system helps students and job seekers understand their current skill gap and provides career recommendations based on mandatory and optional skills.

---

## Features
- User skill management
- Job role and skill mapping
- Skill gap analysis
- Readiness level evaluation
- Career recommendations
- REST API implementation
- Many-to-many relationship handling using JPA/Hibernate
